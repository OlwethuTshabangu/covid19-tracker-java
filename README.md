# COVID-19 Tracker Java Application

Java application developed for the South African Department of Health to retrieve and display provincial COVID-19 statistics from a SQL Server database.

## Technologies Used

- Java
- JDBC
- Microsoft SQL Server 
- Object-Oriented Programming (OOP)
- ArrayList
- Serializable Interface

## Features

- Connects to a SQL Server database
- Retrieves provincial COVID-19 statistics
- Displays total cases and percentage totals
- Uses Java classes and objects to store province data
- Demonstrates database integration using JDBC
